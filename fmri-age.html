fMRI Age Prediction Pipeline

This project predicts participant age from resting-state fMRI connectivity data. The pipeline includes preprocessing, PCA-based dimensionality reduction, and Ridge regression modeling.

Model Performance

  • R² Score: 0.56
  • RMSE: 2.09 years
  • MAE: 1.65 years

Pipeline Overview

The model pipeline consists of:

  1. Loading and flattening fMRI functional connectivity matrices (Schaefer2018 200-ROI atlas).
  2. Scaling and reducing dimensions using PCA (n_components=100).
  3. Training a Ridge regression model with cross-validation to predict age.
  4. Evaluating on a held-out validation set to avoid data leakage.
